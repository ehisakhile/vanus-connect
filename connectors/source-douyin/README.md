# Douyin Source

## Quick Start

### Create the config file

```shell
cat << EOF > config.yml
open_id:
client_key:
client_secret:
EOF
```


| Name          | Required                           | Default | Description |
|:--------------|:-----------------------------------|:--------|:------------|
| open_id       | YES                                |         |             |
| client_key    | YES                                |         |             |
| client_secret | YES                                |         |             |
